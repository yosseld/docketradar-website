Release notes

Everything new, improved, and fixed in DocketRadar. Current version: v0.2.2.

Release history

v0.2.2

Latest February 2026

DocketRadar Rebrand

A complete visual refresh bringing DocketRadar a polished, professional identity built for the legal industry.

  • Complete visual refresh with new brand identity and logo system
  • Updated color system: Navy, Teal, and Slate palette replacing previous colors
  • New typography system using Inter font family
  • Redesigned sidebar UI with improved visual hierarchy and readability
  • New marketing website reflecting updated brand identity

v0.2.1

January 2026

8-County Support

Expanding DocketRadar's coverage from 4 counties to 8, reaching more of Florida's legal community.

  • Added St. Lucie County support (courtcasesearch.stlucieclerk.gov)
  • Added Indian River County support (court.indian-river.org)
  • Added Flagler County support (apps.flaglerclerk.com)
  • Added St. Johns County support (apps.stjohnsclerk.com)
  • Improved county detection logic for faster sidebar loading
  • Updated Free tier to allow selection of any 2 counties (previously limited to specific counties)

v0.2.0

December 2025

Clio Integration

The most-requested feature: direct Clio practice management sync, eliminating double data entry for court updates.

  • OAuth 2.0 authorization flow for secure Clio connection
  • Auto-sync hearings to Clio calendar events
  • Auto-sync docket entries to Clio activity timeline
  • Party information sync to Clio matter relationships
  • Manual sync trigger for immediate data push
  • One-click disconnect with local token cleanup
  • All OAuth tokens stored locally — zero server-side credential storage

v0.1.0

October 2025

Initial Release

The first public release of DocketRadar, bringing automated court monitoring to four South Florida counties.

  • Palm Beach County support (appsgp.mypalmbeachclerk.com)
  • Miami-Dade County support (www2.miamidadeclerk.gov)
  • Broward County support (www.browardclerk.org)
  • Osceola County support (courts.osceolaclerk.com)
  • Sidebar overlay on supported court portal pages
  • Automatic change detection with 15-minute check intervals
  • Chrome desktop notifications for docket changes
  • Case watch list with multi-county organization
  • Local-only data storage — zero data collection
  • Free and Pro tier structure
